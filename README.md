# 关于 MFC-SweepMine

`MFC-SweepMine` 是一款用 `MFC` 编写的 Windows 下的扫雷游戏。是我在 2010 年写的。代码没什么参考价值，但这个游戏还是有点意思的。

所以把代码和可运行程序传上来做个纪念意义吧。

[点此下载程序](https://raw.githubusercontent.com/arnozhang/MFC-SweepMine/master/release/SweepMine.exe)
